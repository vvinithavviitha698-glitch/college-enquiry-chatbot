# College Enquiry Chatbot

This is a simple web-based chatbot developed using Python Flask.
It helps users with college-related enquiries such as courses, fees, admissions, and contact details.

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install Python
2. Install Flask using:
   pip install Flask
3. Run the app:
   python app.py
4. Open browser and visit:
   http://127.0.0.1:5000
